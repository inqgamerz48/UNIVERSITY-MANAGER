## 🕯️ SUMMONING RITUAL (INSTALLATION)

Follow these steps to manifest the project in your local realm.

### 1. Clone the Grimoire
```bash
git clone https://github.com/inqgamerz48/UNIVERSITY-MANAGER.git
cd UNIVERSITY-MANAGER
```

### 2. Awaken the Frontend (Visual Layer)
```bash
cd frontend
npm install
# Create .env.local with your sacred keys
npm run dev
```

### 3. Ignite the Backend (Core Engine)
```bash
cd backend
pip install -r requirements.txt
# Configure .env with DATABASE_URL
uvicorn app.main:app --reload
```

### 4. Open the Portal
 Navigate to `http://localhost:3000` to enter the system.
